Cheat Sheet

for some Ruby

comment #comment the text after '#' will be commented out, and not run if it is on the same line
variable name = 'Anthony' the variable name is set to Anthony
string 'Anthony' Anthony is a string because it is inside of ' ' (quotes)
interpolation "My first name is #{name}" using #{} inside of quotations will generate the value of the variable within the quotes # => "My first name is Anthony"
concatenate "con" + "catenate" two or more strings can be concatenated together # => "concatenate"
